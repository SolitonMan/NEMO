Transaction Validation

This page displays transactions conducted by and for users. If you have the appropriate permissions you can filter by which staff member performed the work, and when, by using the dropdown boxes below.

The table lists all the charges sorted by the most recent Start date. You can filter the table using the search field input below.

Transaction Type Legend

The individual transaction entries have been color-coded and labeled for identification. The transaction types and associated colors are as follows:

  • Tool Use - Light Blue Background
  • Staff Charge - Gray Background
  • Area Access Record - Light Red Background
  • Consumable Withdraw - Yellow Background

Validation Status Legend

Transactions are marked with colored left borders and status badges:

  • Validated - Green border and badge: Charge confirmed as legitimate
  • Contested - Yellow border and badge: Adjustment needed
  • Rejected - Red border and badge: Contest was rejected
  • Open - Gray badge: Not yet validated or contested

The table cells containing notes for individual customers (light green background) or for transactions generally (light blue background) are editable. Click or press Enter on the desired cell to add or edit a comment. Changes are saved immediately to the database.

Each charge can be validated by pressing the "Validate" button on an individual row. Each charge can also be contested by pressing the "Contest" button, which will open a dialog to provide a reason for the contest.
